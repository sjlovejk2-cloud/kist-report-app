# 내부 체크리스트

작성일: 2026-04-16

## 설계서 안전성 판단 기준

- 설계서, 도면, 시방서, 내역서, 견적서의 세트 구성이 있는지 확인
- 총괄표 합계와 세부 내역 합계가 일치하는지 확인
- 수량 × 단가 = 금액 계산 검산 필요 여부 확인
- 공사기간, 위치, 범위, 대상, 제외 범위가 보이는지 확인
- 법정경비와 사후정산 항목이 누락되지 않았는지 확인
- 특정업체 제한 표현, 모델 고정 표현, 1식 과다 여부 확인
- 도면번호, 별첨, 첨부 목록이 있는지 확인

## 향후 앱 고도화 아이디어

- 엑셀 수식 직접 검산
- 총괄표/세부표 합계 비교
- 공종별 체크리스트 분기: 건축, 기계, 전기, 소방
- 법정경비 비율 경고
